# Global Chat Application

A real-time chat application where users from across the world can connect and chat with each other.

## Features

- One-time username popup (persists across refreshes)
- Real-time messaging with WebSocket
- Dark theme with color-coded messages
- Active users list with manual refresh button
- User join/leave notifications
- Emoji support
- Timestamps for all messages

## Local Setup

1. Clone the repository
2. Run `npm install`
3. Run `npm start`
4. Open http://localhost:3000

## Deploy on Render

1. Push code to GitHub
2. Create new Web Service on Render
3. Connect repository
4. Build command: `npm install`
5. Start command: `npm start`
