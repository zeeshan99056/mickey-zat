# Global Chat Application

A real-time chat application where users from across the world can connect and chat with each other. Features include persistent username, active users list, timestamps, and dark theme.

## Features

- One-time username popup (persists across refreshes)
- Real-time messaging with WebSocket
- Dark theme with color-coded messages (green for own messages, light blue for others)
- Active users list with manual refresh button
- User join/leave notifications
- Emoji support
- Timestamps for all messages
- Responsive design

## Technologies Used

- Node.js
- Express.js
- WebSocket (ws library)
- HTML5/CSS3
- JavaScript
